# ECG Heart Pathology Diagnosis

Deep Learning model for classifying heart pathologies from ECG signals using PTB-XL dataset.

## Features
- 12-lead ECG signal processing
- CNN-based deep learning model
- Data augmentation and balancing
- Comprehensive evaluation metrics

## Dataset
PTB-XL ECG dataset from PhysioNet

## Requirements
- Python 3.8+
- TensorFlow 2.x
- WFDB library for ECG reading
